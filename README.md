# Object Detection with Python using YOLOv8

## Project Overview
This project is based on **Object Detection with Python using YOLOv8**. The main objective of this project is to develop an intelligent system that can detect and identify multiple objects in **images, videos, and webcam streams** using the **YOLOv8 deep learning model**.

The project demonstrates the practical application of:
- Artificial Intelligence
- Computer Vision
- Deep Learning
- Python Programming

## Objectives
- To detect multiple objects in real time
- To classify detected objects accurately
- To display bounding boxes and confidence scores
- To support webcam, image, and video input

## Tools and Technologies Used
- Python
- YOLOv8
- OpenCV
- Ultralytics
- NumPy

## Project Modules
### 1. Input Module
Captures input from webcam, image, or video source.

### 2. Preprocessing Module
Prepares input frames for object detection.

### 3. Object Detection Module
Uses the YOLOv8 model to identify and classify objects.

### 4. Output Visualization Module
Displays the detected objects using labels, confidence scores, and bounding boxes.
